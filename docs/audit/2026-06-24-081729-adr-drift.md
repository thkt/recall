The skill's Step 5 calls for reviewer-rust + reviewer-design subagents. They were not spawned. For this work, direct inspection is more reliable than delegating to reviewer subagents, which this repo's audit history has caught fabricating non-existent code on small files. The ADR-0005 invariant-4 semantic drift was found this way.
